About Turing Simulator


Welcome to Turing Machine Simulator.
Upon registering you will be able to create your own machine. Without logging in, you can see simulations for prepared public machines and examples of usage in the Choose simulator tab.
How to create custom machine?
After logging in, you will be able to access Create machine tab. There, you will have to provide some data:
  • Title of the machine
  • Whether or not it's a decisive machine
  • Number of states (without ending state)
  • Alphabet of the machine in format: e.g. a,b,c,d (commas cannot be a part of alphabet)
  • Starting index - where machine should start (insert -1 if it should start from the end
  • Empty sign (default is #)

After creating machine you will be able to download prepared instruction file. You should fill it with instructions. Then upload filled file for your machine. Your machine is now ready to go.
How to create custom example for any machine?
After logging in you will be able to create examples to see simulation for your custom machines and public machines as well. To create example choose available machine from Choose machine tab and click Create example.
Format example: for alphabet [a,b,c,d] insert any combination of available characters, e.g. abbacd
How to see simulation?
To see simulation for a particular example, simply click on the example in the machine in the Choose machine tab. Then press Enter to start machine and Spacebar to move the tape.
